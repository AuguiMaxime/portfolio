# Portfolio — Maxime Augui

Portfolio professionnel de Maxime Augui, développeur web Full-Stack basé près de Toulouse.

## Stack technique

- **Front-End** : React 19, Tailwind CSS v4, Vite
- **Déploiement** : Vercel

## Lancer le projet en local

```bash
# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev

# Builder pour la production
npm run build
```

## Structure du projet

```
src/
├── components/
│   ├── layout/       # Navbar, Footer
│   ├── sections/     # Hero, About, Skills, Projects, Contact
│   └── ui/           # ProjectCard, SkillBadge
├── data/
│   ├── projects.js   # Données des projets
│   └── skills.js     # Liste des compétences
└── assets/           # CV PDF, screenshots
```

## Projets présentés

- **Kasa** — Application de location immobilière (React, React Router, Sass)
- **Mon Vieux Grimoire** — API RESTful de notation de livres (Node.js, Express, MongoDB)
